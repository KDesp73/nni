# nni

NN training interface for the [nn](https://github.com/KDesp73/nn) library

![image](https://github.com/user-attachments/assets/871d6574-7bb3-4484-b5a3-8692bb3c8317)

## Get Started

```bash
git clone https://github.com/KDesp73/nni
cd nni

./configure
make all
```
